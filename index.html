Remote Access

Option 1 — Port Forwarding

▼

Step 1: Assign a Static IP

  1. Open router admin panel (usually 192.168.1.1)
  2. Find DHCP Reservation and assign a fixed IP, e.g. 192.168.1.100

Step 2: Forward the Port

Service: MotorCam  External: 8080
Internal: 8080  IP: 192.168.1.100  Protocol: TCP

Step 3: Find Your Public IP

Visit https://whatismyip.com from the server device

Step 4: Connect

IP: 203.45.67.89    Port: 8080

Option 2 — Ngrok

▼

Step 1: Install & Authenticate

ngrok config add-authtoken YOUR_AUTH_TOKEN

Step 2: Start Tunnel

ngrok http 8080

Step 3: Use the URL

Ngrok gives you a public HTTPS URL to use in Client Mode.

Option 3 — Cloudflare Tunnel

▼

Install & Run

cloudflared tunnel --url http://localhost:8080

Cloudflare gives you a public HTTPS URL instantly.
